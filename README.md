## 🚀 Deploy to Vercel

This repo contains **3 files only**:

```

└── README.md      ← This file
```

Deploy steps:
1. Fork or upload this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → Add New Project
3. Import the repo → Deploy
4. Done — live in ~30 seconds ✅
